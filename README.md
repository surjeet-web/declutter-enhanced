### From ZXP Package
1. Download the latest `Declutter-Enhanced.zxp` from releases
2. Install using ZXP Installer or Adobe Extension Manager
3. Restart After Effects
4. Find Declutter in Window > Extensions menu

1. Clone this repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Enable unsigned extensions in After Effects
5. Load extension from `src` folder
