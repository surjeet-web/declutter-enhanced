Declutter Help

Declutter is your ultimate After Effects organizer! Create custom folder structures, set filters, and sort your assets in seconds. Whether you want to organize specific assets or the entire project, Declutter makes it fast and effortless.

Quick Tips:

  • Choose a Structure: Select a saved structure or create a new one to start organizing.
  • Set Filters: Use powerful filters like composition, audio, video, and more to sort assets automatically.
  • Target or Full Sort: Select specific assets for sorting or let Declutter organize everything in your project.
  • Preferences: Adjust settings like saving your project before sorting or cleaning up unused folders.

Transform chaos into clarity with just a few clicks!
